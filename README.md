# G37OBS
obs-studio plugin for csgo

# Requirement
* Latest version of obs-studio x64, current [25.0.1]

# Features
* Recoil Control System
* Aimbot
* Triggerbot
* Glow ESP
* Bhop
* Interfaces/Virtual Tables
* NetVars/Offsets
* ConVars
* Silent load
* obs-studio game handle
