# G37OBS
obs-studio plugin for csgo

# Requirement
* obs-studio x64 [27.1.3]

# Features
* Recoil Control System
* Aimbot
* Triggerbot
* Glow ESP
* Bhop
* Interfaces/Virtual Tables
* NetVars/Offsets
* ConVars
* Silent load
* obs-studio game handle
